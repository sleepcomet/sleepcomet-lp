## üìñ About

This repository contains the source code for the **Sleepcomet Landing Page**. It serves as the first touchpoint for users, showcasing the platform's capabilities, pricing, and integrations. Built with performance and aesthetics in mind, it leverages the latest web technologies to deliver a smooth user experience.

## ‚ú® Features

- **Modern UI/UX:** Crafted with [Shadcn UI](https://ui.shadcn.com/) and [Tailwind CSS v4](https://tailwindcss.com/).
- **Responsive Design:** Flawless experience across all devices, from mobile to desktop.
- **Animations:** Smooth transitions and engaging effects powered by `tw-animate-css`.
- **Performance:** Optimized with Next.js 16 Server Components and Vercel Analytics.
- **Forms:** Robust form handling with `react-hook-form` and `zod` validation.

## üöÄ Getting Started

We love contributions! Follow these steps to set up the project locally.

### Prerequisites

- Node.js 20+
- npm or pnpm

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sleepcomet/sleepcomet-lp.git
    cd sleepcomet-lp
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
